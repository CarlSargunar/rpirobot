# Create Raspberry Pi Robot Project #

- Enter the Share Directory;

    ```
    cd /home/pi/share
    ```

- Create Pi Robot Project;

    ```
    dotnet new console -o rpirobot
    ```

- Enter the Pi Robot directory;

    ```
    cd rpirobot
    ```

- Test the project runs

    ```
    dotnet run
    ```
